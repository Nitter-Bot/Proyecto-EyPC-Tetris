# Proyecto-EyPC-Tetris
Proyecto en Assembly para Estructura y Programacion de Computadoras implementando el juego de Tetris

## Plantamiento 

* Diseñar y programar en lenguaje ensamblador, con instrucciones enfocadas
en el procesador 8086, un programa que haga uso de:

    * Bucles

    * Entrada por teclado

    * Manejo del teclado

    * Entrada por mouse

    * Salida por pantalla

    * Menús en modo texto

    * Operaciones aritméticas binarias y conversión ASCII <- ->  binario
