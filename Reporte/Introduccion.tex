\section{Introduccion}

 Para el proyecto de Estructura y Programacion de Computadoras se
 implementara un juego de Tetris en Assembly x86 con el fin de mostrar
 el dominio de

\begin{enumerate}
	\item Bucles

	\item Entrada por teclado

	\item Manejo del teclado

	\item Entrada por mouse

	\item Salida por pantalla

	\item Menús en modo texto

	\item Operaciones aritméticas binarias y conversión ASCII <- ->  binario

\end{enumerate}
